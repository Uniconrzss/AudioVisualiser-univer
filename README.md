# AudioVisualiser
An FFT bar audio visualiser written in python.

USAGE '~/python3 fft.py (THEME) [OPTIONS]'

------------------OPTIONS------------------
#   -h Displays this help menu
#   -m Mono Mode
#   -s Stereo Mode
#   -S Stereo With Average
#   -l Channel Left (mono)
#   -r Channel Right (mono)
#   -a Channel Average (mono)
#   -M Max Smoothness
#   -A Avarage Smoothness
#   -H High Smoothness
#   -L Low Smoothness
